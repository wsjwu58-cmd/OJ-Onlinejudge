# OJ-OnlineJudge

#### Description
本项目是一个功能完善的在线判题系统（Online Judge)，采用前后端分离架构，集成了开源沙箱，支持代码提交、自动判题、AI辅助判题、竞赛管理、题组管理等功能。系统集成了先进的AI技术，实现了AI判题、错误分析、智能提示等创新功能。
